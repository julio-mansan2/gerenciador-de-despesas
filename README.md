<br>

- Utilizar datas no JavaScript:

````javascript

const date = new Date();
const weekDay = date.getDay();

````

- Aplicar estilos a elementos:

````javascript

if (weekDay === 0) {
        graphic[6].style.background = 'hsl(186, 34%, 60%)'
        graphic[0].style.background = 'hsl(10, 79%, 65%)'
````
